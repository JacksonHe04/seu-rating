苏打绿 Sodagreen

成立日期: 2001年4月26日 更多中文名: 蘇打綠 更多外文名: oaeen 厂牌: 林暐哲音乐社 流派: 流行 语言: 普通话 乐队成员: 吴青峰 / 谢馨仪 / 史俊威 / 龚钰祺 / 何景扬 / 刘家凯 职业: 音乐人

苏打绿（Sodagreen）是一个台湾的乐团，于2001年由国立政治大学的学生组成。2003年7月，苏打绿在海洋音乐祭小舞台演出，在此次活动中结识音乐制作人林哲，而后加入林哲音乐社，陆续发行三张单曲，并于2005年9月发行第一张同名专辑《苏打绿》。2003年时确立了了现在的六人阵容：主唱吴青峰（青峰）、贝斯手谢馨仪（馨仪）、鼓手史俊威（小威）、电吉他手刘家凯（家凯）、木吉他手何景扬（阿福），以及键盘与中提琴手龚钰祺（阿龚），其中除了阿龚为台北艺术大学音乐研究所的学生外，其余五名成员都曾就读政治大学。自出道开始，苏打绿所有发表歌曲皆为团员所创作。
